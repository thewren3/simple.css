About

The Upper Valley Running Segment Challenge is based off of a similar challenge that happened in January, where runners in different cities competed against each other by running a specific “segment” as many times as possible. In one city, the winner ran 450+ miles on a .22 mi segment, running that segment more than 2000 times! You can read more about this challenge here – it was very fun to follow along! I thought that this challenge would be fun to bring to the upper valley as it rewards distance and time on feet, not speed, which is a fun twist on racing.

For our Upper Valley edition, we will adapt the rules a little to encourage wider participation. But the gist of the competition is: every segment completion nets you 1 point. Whoever has the most points at the end of the competition wins.

Everyone is welcome and encouraged to participate! There is no cost to entry. You do not need to have Strava; all you have to have is some way of tracking your run (Garmin, nikerun, apple, coros, etc.)

How it Works: After any run in which you've completed at least one segment, submit that run and your segment completion(s) will reflect on the leaderboard the next time it is updated. Submissions should be done within 1 week of completion (to make sure the leaderboard is at least somewhat up-to-date)

The Segments

There are 6 segments in the challenge, designed as such to maximize accessibility. All of these segments count for the same, which means that each of these, when completed, count for 1 point. Each segment is roughly .4 miles, and roughly flat.

Segment 1: Pine Park River

Segment 2: Beaver Meadow by Huntley

Segment 3: Rail Trail dirt

Segment 4: Along Hartford ave. under 91

Segment 5: Downtown WRJ

Segment 6: Mt. Support by SOJ

Rules

Competition Guidelines:

  • Segments are only one direction. There is an established start point, and an established finish point. So if you complete a segment and then run that segment back the way you came, this would still only count as one segment completion.

  • One segment completion equals one point. You may complete a segment as many times as you want within a run (and thus you can net multiple points all within one run). You may also complete multiple segments in one run. You do not need to run all 6 segments -- in fact, you could win the entire competition by only running one segment (a lot of times).

  • Segment completions must be done on foot. No vehicle assistance allowed.

  • Segment completions have to be proven; to prove you’ve finished a segment, you can either submit a link to the strava/garmin/coros activity, or link to a shared google doc with a picture/screenshot of the run details. Docs should be shared with uppervalleysegmentrunning@gmail.com

  • Segment completions should be submitted within 5 days of completion. In the last week, May 24th-May 31st, submissions should be within 24 hours of completion.

  • Anyone and everyone is welcome to participate! Spread the word to your friends, family, and neighbors!

  • This challenge’s time frame is from May 1st to May 31st, 11:59pm
